Objetivos:

O objetivo do Organiza! é criar um site para oferecer um apoio para os movimentos comunistas brasileiros.
assim incentivando a entrada de novos participantes e facilitando dos que ja participam organizando o movimento.

Ideias:

Como ideias para serem implementadas dentro do blog tem um sistema de nodificação dos movimentos que vão acontecer dando
data,horario e local, um sistema de postagem das ultimas noticias dos movimentos e criar um chat para comunicação com
uma instituição brasileira como exemplo o PCB.
